# 我是如何在生财9个月写了9篇精华｜老黄牛

# 一、如何通过【卡片法】学习、思考、写作

第一个部分是关于：学习、思考、写作

看上去项目比较多，但其实都是关于写作。写作不是输出，而是“输入+思考+输出”

如果说阅读帮助你与很多「已逝的伟人」交朋友。那么写作帮助你与很多「不在身边的人」交朋友。你的文章是你的名片，引起与你频率相同人的共鸣，产生了更多让陌生人认识你的机会。

我进入生财有术半个月以来，就很想写文章，然后苦学了很多写作技巧。直到学习了【卡片法】，让我打开了新世界大门。

我通过一篇文章，第一个月就进入“龙珠俱乐部”然后认识了不少现在的朋友，很多为我以后做投资、工作、生活上提供了很好的建议，改变了很多。

# 1、如何写作

普通人写作是这样的：安排一个时间，倒一杯茶，坐在电脑前，然后拿着之前想到的主题，在电脑前各种找资料。最后弄半天写不出来，放弃了。

「卡片写作法」是这样的：平常生活刻意累积卡片，当卡片积累到一定程度，就开始写作。文章的标题是根据卡片的内容而定的。

你会发现两者是不一样的，「卡片写作」是根据有什么材料，再决定做什么菜。

因为写作不是输出，写作是【输入+思考+输出】。

没有输入，无法做好的输出。

# 2、如何输入

普通人输入是这样的：微信公众号看到好的文章，然后点击收藏。在“得到”“kindle”或者“微信读书”看到好的句子，然后划个线。

# 「卡片输入法」

①收藏文章并不等于学到知识，只是知识的搬运工。

如果我们用文章比喻成大厦，那知识卡片就是砖头。一篇文章就像已经建好的大厦，华丽但对你写作无用，当你要用到里面的知识点的时候，需要重头看一遍。

我们应该积累的是卡片，而不是文章。

②卡片要归一

如果你在“微信读书”看好很启发、有用的句子，可能就划个线，写一下自己的感想。

但最重要的是要把他们抄下来，放到同一个的APP。要不然你会忘记这个想法来自于哪个信息源，而无法找回它。

推荐用“flomo”

# 3、如何写一张卡片

# 写一张卡片非常简单，4个模块

①原文句子

②来源

③自己的想法

④与它相关的想法

# 让卡片之间产生连接

①光积累卡片是不够的，还要用卡片连接起来。

卡片只是一个节点，卡片之间的连接就会产生新的想法，新的文章。

有个比喻很好：卡片是城市，连接是高铁。文章是整个系统。高铁没新增城市，但是让被连接的城市更繁荣。

②如何连接

有空就要多回顾积累卡片，然后把有关联的卡片做个连接。

一张卡片大概就是这样：

硅谷创业教父 Paul Graham 在 《如何写作》开篇所言：

写作不仅交流想法，更催生想法。但大部分人，终其一生都不能意识到好的写作竟如此重要。如果你不善写作，也不爱写作，那么，你在余生将与贫瘠思想为伍，错过经由写作带来的新世界。

好的写作，可以破圈入龙珠俱乐部，可以获得工作机会、连接机会。各位可尝试卡片写作法，多积累卡片，是有复利效应的，无论是写作能力，还是人生际遇。

# 二、如何通过【卡片法】辅助创业

很多为网赚副业项目，技巧都是相同的。例如私域流量。做淘宝、天猫、社群。。。万物皆可私域。例如低成本获客，也有很多种方法。

我统一把这些方法叫做【技能卡片】吧**。**

大家看生财的时候，看到文章里有适合自己，或者厉害的方法，可以先用卡片存起来，以备不时之需。

举个例子：什么是好【技能卡片】。「生财合伙人」嘉宾@Kosan，写了一篇文章——《从创业失败到低成本获客过百万，我是如何摸索出自己擅长的引流打法的》 (opens new window)

里面提到：

【每年国考公告出来，一定是会上微博热搜，也是一定会上知乎热门的。国考的公告时间公布不确定，但肯定每年都会公布一次。于是我每年就提前 1 个月准备好一个问题 + 一个回答。等国考公告一公布，我就马上去知乎提问 + 第一个回答。

我的提问都特别简单，每年都是「如何看待20**年国考公告」，然后我就把早就准备好的文章更上去，然后就去邀请全站人回答，就可以了。可能很多人都想不到，竟然这么容易，但它就是这么容易，而且，至少在过去 2 年，我们整个行业，就我一个人在干这事】

所以卡片我们就可以这样写：

搜微博热点或者重复性节日，准备一个问题+一个回答，发知乎，邀请全站人回答。

【每年国考公告出来，一定是会上微博热搜，也是一定会上知乎热门的。国考的公告时间公布不确定，但肯定每年都会公布一次。于是我每年就提前 1 个月准备好一个问题 + 一个回答。等国考公告一公布，我就马上去知乎提问 + 第一个回答。

我的提问都特别简单，每年都是「如何看待20**年国考公告」，然后我就把早就准备好的文章更上去，然后就去邀请全站人回答，就可以了。可能很多人都想不到，竟然这么容易，但它就是这么容易，而且，至少在过去 2 年，我们整个行业，就我一个人在干这事】@Kosan

原文：https://articles.zsxq.com/id_stwpm0zksnww.html (opens new window)

生财里面各种“八仙过海”的技巧非常多，把文章里的【技能卡片】积累下来，而不是把文章收藏起来。

# 三、如何通过【卡片思维】防止被割韭菜

如何防止被骗，或者被割韭菜？

其中一个维度：看他的文章有没【技能卡片】，一篇文章里，可收藏的【技能卡片】越多，含金量越高。逻辑清晰，可信度越高。

通常镰刀的文章是这样的：

作者头衔很厉害，喜提高铁飞机，显得赚很多。自己项目能赚很多，亲戚朋友跟着作者就能赚钱。

但发现整篇文章，里面无法找到一个可收藏的【技能卡片】，为何能赚钱没有说清楚。

头衔和过去的业绩当然可以侧面判断一个人或一个项目是否成功，但如果没有点说服人的逻辑或可收藏的技能卡片，那就留个心眼了。

这也是我在投资领域训练回来的触觉：收益不重要，逻辑最重要。

就好比有人赌场赌大小，连续赢了2盘，然后说上年收益400%，这是没有意义的。看就是这个东西是否可重复，收益长期做是否为正。

有句话是这样“我知道你的立场了，那你的逻辑与数据呢？”

关于我觉得非常好的一篇文章，就是里面满满的【技能卡片】，但作者无贴任何标签。

大家可以亦仁写的这篇——《如何找到1000个小思路》 (opens new window)

# 总结：

1、多输出能帮助你与很多「不在身边的人」交朋友，可以破圈拓展人脉，增加机遇。如何多输出呢?不是坐在电脑前憋一天，而是平时多注意收集卡片。卡片越多，输出越容易。

2、先模仿，再超越。多收集【技能卡片】，灵活运用到自己的项目中。

3、一篇文章，一个项目，只有炫耀，没有【技能卡片】，没有逻辑，多留个心眼。
