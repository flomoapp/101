# 记录自己的思考和日常太治愈！｜饭饭

今年1月份的时候，遇见了flomo笔记，之后写了一篇《能让你产生心流感觉的笔记软件 (opens new window)》（没看过推荐先看这篇），到今天，我也依然认为，它帮我抓住了更多生活中的心流时刻。

同时，用了7个多月之后的今天，我惊喜的发现，它（或者说记录思考和日常这个习惯）帮我建立起了与自己内心对话的桥梁（更少的抗拒，更多的自我接纳），以及对生活更有实感。

之所以会惊喜，是因为**做的过程很轻松甚至愉悦，却还给我带来了这么多的收获。**在这个总是让人非常疲倦的时代，无比珍贵。

# 01.通过标签发现独特的自己

和很多朋友推荐了flomo后，我们时不时会分享各自有意思的新标签，特别有意思。

看多了就发现，从flomo标签中就很可以看出这个人在关注什么、喜欢思考和记录什么，这点每个人会有非常大的不同，都非常有个人特色。

这也是最有意思的地方，我之前看过很多印象笔记笔记的分类文件夹，几乎都大同小异，每个人都是按照那些差不多的分类往里面填充内容。

但在使用flomo的时候，没有文件夹，而是在写的过程当中，不断建立标签。也就是说，这些标签是记录的过程中，慢慢生长出来的。

因为标签都是自己自然而然写出来的，喜欢就自然会更愿意多写些，不知不觉就会有些写得频率比较高的标签出现，这很可能就是我们真正关心和喜欢的点/领域。（很喜欢这种不知不觉间自然涌现出来的觉察，因为这样更接近真实。）

发现自己喜欢的领域很重要，时间和自我管理中有个重要的法则，就是要把自己的时间和注意力用在自己喜欢且真正重要的事情上。

在这类型事情上花的时间越多，内心就越踏实。

# 02.小小的闪念发展出大大的人生

每次看书、交谈、散步的时候，经常会冒出一些灵感和闪念，以前会觉得这些灵感和闪念太小，不值得记录。（这是个极大的误解）

现在因为有了flomo这个专门用来记录这类闪念、而且可以帮我很轻松管理和安放自己闪念的笔记工具，就很愿意随时随地记录下来。

忘记是在村上春树的哪本书上看到的（好像是《我的职业是小说家》），他说：

年轻的时候灵感很多，写作的时候随便一抓就是一大把。但是到了后期，需要每天坐在书案前写个几小时，才能保证自己总是能抓住那倏忽而逝的一点灵光。\

你看，即使是最优秀的写作者，其实不断在练习的也是记录和写出自己灵感的能力。

我一直觉得，每个人都可以成为好的**「写作者」**，都会有很多的灵感，区别只在于，你有没有把你的这些灵感记录下来。

如果你总是忽视自己的灵感，灵感就会越来越少，直到你再也看不见它。

可一旦你建立了记录自己灵感的习惯，接下来就是一个**「手熟尔」**的过程，你会越来越擅长做这件事情，以至于你可以写得比绝大多数人要好。

我自己就是这么过来的，几乎没学过什么写作技巧，都是凭着直觉去写作的，所以文笔各方面的很一般，甚至有很多语法错误。

但这么多年写下来，几乎每篇文章都会有读者和我分享他们阅读之后的收获，让我知道我写的东西是有价值的，这就足够了。

到现在，我越来越熟练与做这件事，好多次都是只是想简单记录下，结果写着写着，一口气写了一大篇出来，稍微改改就可以发布了。

收获了太多太多的激励和祝福，感恩~\

# 03.记录生活小片段实在是太治愈了

我的flomo里有好几个记录日常生活的标签，比如：#梦、#情绪日记、#记录日常，里面的内容也大多数都是日常生活的鲜活记忆点。

我以前的写作和思考都是围绕着工作、目标、财富、成长这些，总是无意识的忽视了思考和记录生活。

但现在最治愈我的就是把一些有意思的生活小片段记录下来。

有个经常发生的场景就是：

以前坐地铁都是看文章看视频，现在经常是因为之前朋友的一句话或者想到一件事，而产生了很多闪念，在手机上噼里啪啦的打字记录自己的思考，3-40分钟的坐地铁时间一下子就过去了，然后还写了很多思考，这种感觉实在是太好了。

因为这些标签，最近明显感觉到，我对生活更有实感了，越来越能在现实生活中获得乐趣。

# 04.高度的自我觉察带来高频的自我接纳

开始高频记录之后就发现，我时常会在边记录的时候边想通很多事情，有些是对自己的，有些是对生活的。

经常是吃个饭、坐个地铁，都能发觉到自己的很多个性和独特的反应，这是一种高度觉察的状态，但又很随意和轻松，至少比冥想和禅修时候要轻松地多。

而且大概是因为这种自我觉察的时刻变多了，最近这个几月时常就会有变得自洽和自我接纳的感觉。在各种事情上，突然就不纠结拧巴了，日常的幸福度变得极高。

不知道大家能不能感受到我的这种愉悦，最关键的是，这些都是自然而非发生的，我并没有说很努力看很多书、做很多刻意练习，就是这么不经意间就获得了。

# 05.虚度时间的感觉变少了

平时我们总是感觉时光飞逝，不知不觉一天、一周、一个月就过去了，回想的时候却怎么也想不起来这个月做了哪些事情。

但有了记录自己日常闪念的习惯，每天写个3-5条，这些就像时间凝结出来的一个个果实一样。

下次再回顾自己过去的时间，就可以翻看这些笔记，就不再是缥缈的记忆（好像什么都没做的不安感），而是手握大把时光果实的充实感。

# 写在最后：

今天分享了很多写flomo的好处，但不知道大家有没有发觉，这背后还有一个很关键的共通点，那就是：实现这些好处都是「自然不费力」的。

自由职业之后，一直在建立自己的日常流程（学习、工作、锻炼、饮食作息），其中最重要的一条就是愉悦地完成所有的事情，消除各种不必要的内耗。

愉悦的做并不是说没有困难，不付出辛苦，而是内心**「不抗拒」**。如果内心抗拒，需要逼自己去做，就算做完还要面对因此产生的负面情绪，这样就非常浪费能量。

所以，每次出现内心抗拒的事情，我并不是拖延也不是动用自律去让自己做，而是反复思考，内心抗拒的原因，然后找到更适合自己或更喜欢的方式达成想要的目标。

《卡片笔记写作法》中有句话我一直记得：

高效率的人不会与不利的阻力作斗争，而是像柔道冠军一样转移阻力，这不仅要依靠正确的心态，也要依靠正确的工作流程。

只有内心不抗拒，做起事来才会愉悦，也会更容易专注于投入获得心流。某种意义而言，幸福感就是心流时候体验到的感觉。
