# 📆 每日回顾

本功能需 PRO 会员权限→

flomo 有 3 种方式帮你回顾过往的 MEMO ，让你过去记录的内容不被遗忘；也让知识碰撞，激发新的灵感。

  • 方法 1：通过桌面小组件进行回顾（每天最多 24 次）
  • 方法 2：通过 App 的推送消息，在通知中心回顾过往内容（每天最多 24 次，iOS 暂不支持）
  • 方法 3：通过微信服务号，每日推送消息进行回顾（每天最多 3 次）

你可以根据需要选择回顾的范围，所有回顾是纯粹的随机。参考用法：如何更好地回顾

# 一、通过桌面小组件回顾

# 功能说明

  • 你可以在 iOS/Android 桌面上添加每日回顾小组件（如果付费会员购买后依旧提示，请重新添加。部分功能 iOS 正在开发）
  • 可以在每日回顾 - 小组件回顾中设置对应条件及回顾频次。

查看视频教程：笔记如何设置每日回顾？ (opens new window)

# iOS设置说明

如果想要设置回顾内容（如图四），需要点击 flomo App 左上角菜单 - 每日回顾 - 小组件回顾。

如果想回顾更多，可以添加两个尺寸不同的小组件。

注意：设置小组件之前，请先登录flomo帐号，否则PRO 会员会提示无法回顾，但其实问题是没有检测到登录状态；iOS 如果始终搜索不到，请更新 iOS 版本至 14.5 以上。

# Android设置说明（以华为手机为例）

由于各厂商设置不同，无法一一说明，大概操作步骤为

  1. 长按桌面空白处（或者双指向内捏合），进入选择模式
  2. 选择桌面小组件（或者窗口小工具）
  3. 找到 flomo 小组件
  4. 选择想要的组件样式，添加至桌面。有 2 种不同的小组件（输入/回顾），5 种不同样式
  5. 调整小组件参数（颜色模式、字体大小、不透明度），保存并返回桌面
  6. 在 App 左上角菜单 - 每日回顾 - 小组件回顾中，可以设置回顾内容范围

# MIUI 设置说明

相关设置在 App 左上角菜单 - 每日回顾 - 小组件回顾中（同 iOS ）

# 二、通过 App 推送回顾（目前仅支持安卓）

功能说明

  • 在系统设置中打开 flomo 消息推送，就能收到每日回顾的消息推送
  • 注意：请打开 App 的自动启动权限，确保推送能准时送达

操作步骤

  1. 在系统设置中打开 flomo 消息推送
  2. 点击 App 左上角菜单 - 每日回顾 - 推送回顾，点击开启

# 三、通过微信服务号回顾

# 功能说明

  • 当你关注 flomo 服务号之后（微信搜索 flomoapp），开启微信回顾可以每天通过模板消息提醒回顾。
  • 每个账号能设置最多三条提醒，每条提醒可以设置独立的条件和提醒时间。

  1. 点击 App 左上角菜单内的【微信输入】，获得微信专属二维码，发送至微信。长按二维码，关注公众号后，将自动绑定 flomo 帐号。
  2. 点击 App 左上角菜单栏的【每日回顾 - 微信回顾】，添加回顾规则（最多 3 条），每条提醒可以设置独立的条件和提醒时间。

# 多任务界面锁定

在「多任务管理页」中，找到「flomo笔记」，按住卡片向下滑动，出现锁定标识即可。

# 允许自动启动

在「设置 - 应用和服务 - 应用启动管理」页面，找到「flomo笔记」，在「启动管理」中设置为手动管理，允许自启动、关联启动、后台活动（放心，flomo 不会乱唤起其他进程）

# 关闭电池优化

在「设置」中搜索「电池优化」，找到「flomo笔记」，设置为「不允许」（放心，flomo 并不费电）

# 开启通知权限

在「设置 - 通知」页面，找到「flomo笔记」，开启「允许通知」

# 关闭省电模式

在「设置 - 电池」，关闭「省电模式」，「超级省电」
