# 🏷 使用标签

flomo 的标签系统比较特别，不但支持层级，还能在正文中当做内容使用。

在传统的文件夹模式下，「电动按摩椅」要么被放在「家用电器」中，要么被放在「家具」中，相互互斥。而标签的好处在于，你可以在任意一个 MEMO 上添加多个标签，方便你将来的回忆和查找。

关于标签的使用理念详见 I.A.R.P 标签管理方法，这里着重讲述的是操作方法：

注意：为了降低服务器压力，标签并不会立即同步到所有平台，如果你在 A 平台（比如 Mac 上）新增了标签，那么请重启下 B 平台（比如手机）后才会同步

# 如何新建标签

flomo 的标签格式为 #标签 。多个标签之间记得加空格。

在 flomo 输入时，你可以随时通过点击 flomo 的输入框下方的「#」按钮，或通过键盘直接输入「#」号，来插入一个标签，例如 #标签 （记得以空格结尾）。

输入完成后点击发送按钮，即可看到标签创建成功。至于标签放在开头还是结尾，以及是否放在 MEMO 中间，这个并无标准，顺手就好。

注意：
请不要在标签中使用 emoji 以及各种特殊字符，会导致无法查询。如#😁开心或#读书&学习。
请不要用 #日记# 这种以及#结尾的格式，后面的标签会导致无法查询。
如果有英文单词，之间请用连接符如 #Apple_seed 标签与标签之间，需要有一个空格分割，否则会被识别为一个标签。

# 如何新建多级标签

flomo 支持 #标签/子标签 这样的多层级方式，这样既能兼顾标签的便利性，又能方便浏览。比如我们读了一本书《Different》，想把它放在 #读书标签下，并且还保留几个书中的章节，那么我们就可以写作：

#Books/Different/营销近视症这样就相当于创建了三个层级的标签

# 标签如何排序

目前 flomo 的标签按照字母表顺序排序。

如果想要修改顺序，可以在标签前面增加数字，如从#标签改为#01_标签。

拖动和其他排序方式尚在思考，请诸位耐心等待。

# 如何修改或置顶标签

置顶标签已经全部支持，修改标签目前除了 iOS 版本均已支持。

  • 如何置顶：点击标签右侧的...，根据提示点击，即可置顶。最多置顶 10 个标签。

  • 批量修改：点击标签右侧的...，根据提示点击修改即可。如需修改 MEMO 的标签，进入编辑模式删除对应内容即可。

注意：修改标签不仅能修改名称，还能修改标签层级。比如想要把 #幸福 放在 #人生 这个标签下面，就可以直接修改为 #人生/幸福

# 如何删除标签

目前 flomo 不支持从侧边栏批量删除标签，这样一旦误操作代价太大。如果你想删除某个标签，可以参考下面的方式。

  • 单独删除：直接编辑 memo，删除对应标签内容即可
  • 批量删除：暂不支持，但可以通过修改标签把内容合并到某个标签下

# 如何找到未添加标签的内容

在网页端，点击搜索框即可弹出「无标签」搜索选项

# 如何在标签中使用 Emoji

使用 Emoji 可以让的标签更加醒目和特别。在 flomo 的网页端和安卓端，目前支持点击标签前面的图标进行更换。仅限 PRO 用户使用。
