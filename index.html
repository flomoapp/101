# flomo 浮墨笔记是什么

flomo 卡片笔记，是百万学习者的口碑之选，专注在碎片化时代，促进你的记录，帮你积累更多知识资产。

和传统笔记工具不同，flomo 的特色为：

极简记录

  • 像聊天一样记录，无需排版布局
  • 支持微信输入 及诸多第三方插件输入

云端同步

  • 免费支持全平台同步（PC/Web/App），不限制设备登录数量。

回忆更好

  • 支持多种手机小部件、推送回顾，和过去不期而遇。
  • 支持通过微信回顾，无需打开 App 即可查过往笔记

便捷整理

  • 使用多级标签管理记录，灵活又便捷
  • 利用批注建立卡片之间连接。

专注个人用户

  • 终身无广告，无社交，聚焦个人知识管理。

# flomo 能干什么

这是根据我们大量的调研，整理出来的大家经常拿 flomo 做什么的场景：

  • 积累知识： 如建立读书笔记、课程笔记、论文资料等
  • 沟通记录： 如会议纪要、同事讨论、聊天记录等
  • 个人思考： 如日报周报记录、自己当下遇到的问题和灵光一闪的想法
  • 情绪日记： 记录自己的情绪、小确幸等，让自己获得平静

但在 flomo，我们不仅仅是设计工具，也希望能给你带来全新的思考方式。

所以在使用 flomo 的同时，查看关于 flomo 思维方式 和 常见用法，能让你使用时对自己产生更大的价值。

# flomo 诞生的契机

工作这些年来发现， 多数人没有记录想法的习惯。
不是因为他们懒惰， 而是今日的工具和方法， 都进入了一个误区：
或者在聚焦如何设计更好的「写作体验」 或者设计更丰富的「建立知识之间的连接」

而大多数笔记产品都忘了， 能成为作家的人少之又少， 而需要思考的人却越来越多
相比如何组织知识和写出文章
我们多数时候缺乏的是更多的记录，以及通过记录引发的思考

麦克卢汉说：我们塑造了工具，尔后工具也在塑造我们

从这个角度来看，
flomo 没有 Notion 强大排版功能
亦没有 Roam 精妙的双向链接
比不上 Typora 的 markdown 编辑器

因为 flomo 只朴素的关注一个事实
能否帮你记录更多的想法，让你的记录习惯更加持久
帮你积累知识的复利

  • Andy Matuschak (opens new window)，iOS GUI 设计师，可汗学院领导研发工作，EvergreenNotes 概念提出者

# flomo 不擅长什么

我们深知做好一件小事就已经很不容易，所以不会试图在功能层面 All in one，如果你对 flomo 有下面的预期，那么可以选择我们推荐的其他产品替代。

  • 不擅长写 文档撰写，推荐石墨文档 (opens new window)
  • 不擅长做 文档收藏，推荐印象笔记 (opens new window)
  • 不擅长做 TODO，推荐滴答清单 (opens new window)
  • 不擅长做 思维导图，推荐 XMind (opens new window)
  • 不擅长做 助记卡片，推荐 Anki (opens new window)
  • 不擅长做 电子表格，推荐 Airtable (opens new window)

另外我们很敬佩 Notion (opens new window)、RoamResearch (opens new window)，因为他们给了 flomo 很多启发。
